Sobre a Code for All

Nossa História

A Code for All nasceu em 2015 com a missão de usar a tecnologia para transformar comunidades. Começamos com um pequeno grupo de desenvolvedores apaixonados por impacto social e, desde então, crescemos para uma rede nacional de voluntários dedicados a criar soluções tecnológicas para desafios sociais.

Nossos Valores

Acreditamos em colaboração, transparência e impacto mensurável. Valorizamos a diversidade e inclusão em nossa equipe e projetos, e nos esforçamos para criar um ambiente onde todos se sintam bem-vindos e valorizados. Nosso trabalho é guiado pela ética e responsabilidade social.
